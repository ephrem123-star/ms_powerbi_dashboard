# Big Mart Sales Prediction Dashboard

This repository contains a **Power BI dashboard** for analyzing and predicting sales of Big Mart stores using the popular Kaggle dataset. The dashboard provides insights into sales trends, store performance, and product categories.

## Dataset

The dataset used is from [Kaggle: Big Mart Sales Dataset](https://www.kaggle.com/competitions/big-mart-sales-prediction/data). It includes:

- Store information
- Product details
- Historical sales data

## Features

- **Sales Analysis:** Explore total and average sales by store and product category.
- **Sales Prediction:** Forecast sales using historical trends.

## Usage

1. Open the dashboard PDF to view sales insights.
2. For full interactivity, open the Power BI `.pbix` file (not included in this repository).
